Colo Colo
=========

A [flying chicken](https://github.com/codeforamerica/flying_chicken) client
that uses [Growl](http://growl.info/) to notify you of GitHub commits.

## Usage

Install depencies with

    $ npm install

Then run with

    $ node app.js &

## Q + A
